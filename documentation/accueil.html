Généralités sur GénéGraphe

Ce logiciel vous permet de dessiner des arbres généalogiques à partir des données saisies dans le logiciel Généamania. Vous pouvez aussi placer des étiquettes, ce sont des zones de texte dont vous pouvez définir la présentation (couleurs et formes).

GénéGraphe utilise tous les renseignements saisis dans le logiciel Généamania mais ne modifie rien dans cette base de données. Par exemple, quand vous supprimez une personne d'un arbre dans GénéGraphe, la personne existe toujours pour Généamania. Elle peut de nouveau être placée dans un arbre GénéGraphe. Autre exemple. Lorsque vous changez le nom d'une personne ou une date dans Généamania, il suffit d'ouvrir l'arbre dans GénéGraphe qui contient cette personne et la mise à jour est prise en compte.

Quand un arbre vous semble achevé, vous pouvez générer un fichier PDF et/ou générer des images contenant cet arbre. Généamania vous permet d'afficher les arbres créés par GénéGraphe. Vous pouvez aussi générer une fiche par personne dans un fichier PDF.

Bien entendu, vous pouvez sauvegarder vos arbres et les reprendre pour les modifier. Ces arbres sont sauvegardés dans la même base de données que Généamania.

Pour que les arbres générés conservent une homogénéité de présentation, GénéGraphe permet de définir des modèles d'étiquettes pour les personnes et pour les unions. Les modèles définissent les valeurs qui s'affichent pour une personne ainsi que la mise en forme du texte affiché. Vous pouvez définir autant de modèles que vous voulez.

GénéGraphe permet de définir des préférences de présentation. Elles sont appliquées dès que vous ouvrez un arbre et modifient la présentation de vos arbres en conséquence.

Dans ce document, je parle d'objet. Ce terme désigne une personne, une union ou une étiquette.
