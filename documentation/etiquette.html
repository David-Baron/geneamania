Etiquettes de texte

Ajout d'étiquette - Modifier le texte - Dimension d'une étiquette - Présentation d'une étiquette

Les étiquettes sont des zones de texte dont vous pouvez modifier la présentation. Elles servent à mettre un titre ou à donner une information complémentaire.

Ajout d'étiquette

Cliquez sur le bouton , une étiquette s'ajoute à l'arbre.

L'étiquette est le rectangle noir. Pour le moment, elle est vide.

Modifier le texte

Quand vous cliquez sur le bouton , le logiciel change de mode de travail. Il vous permet de modifier le texte de toutes les étiquettes et il interdit toute autre possibilité (ajout de personne, déplacement d'objet…). Pour bien indiquer le mode de travail, le bouton a changé d'icône, il se présente maintenant comme ceci : .

Pour changer le texte d'une étiquette, cliquez dans l'étiquette, vous pouvez saisir le texte que vous voulez. Le fait de cliquer dans une étiquette fait apparaître une barre d'outils d'édition de texte.

Vous trouvez, dans l'ordre :

 choix de la police de caractères
choix du corps (taille) des caractères
alignement à gauche
alignement au centre
alignement à droite
couleur du texte
mise en gras
mise en italique
soulignement du texte
coupe le texte sélectionné dans le presse-papier
copie le texte sélectionné dans le presse-papier
colle le contenu du presse-papier à l'emplacement du curseur

 

Vous pouvez utiliser les raccourcis habituels pour copier du texte ( + ), coller du texte ( + ) ou couper du texte ( + ).

Quand vous avez modifié le texte de toutes les étiquettes, cliquez sur le bouton pour revenir au mode de travail sur les objets.

Dimension d'une étiquette

Quand une étiquette est sélectionnée, elle est entourée d'un cadre bleu. De plus, elle dispose de 8 petits carrés. Ce sont les poignées qui permettent de modifier sa dimension.

Présentation d'une étiquette

Un chapitre entier est consacré à ces possibilités.
