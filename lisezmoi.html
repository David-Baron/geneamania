Première installation de Généamania

Etape 1 : installation du logiciel en local :
  • Version autonome :
    Décompressez l'archive de la version autonome sur votre disque dur ou sur une clé USB (évitez les répertoires avec des accents et la racine).
  • Version non autonome :
    Décompressez l'archive de la version non autonome sur votre disque dur dans le répertoire des sites web.
Etape 2 : initialisation en local :
  • Version autonome :
    Cette phase est sans objet ; passez directement à l'étape 3.
  • Version non autonome :
    NB : l'utilisateur, à priori expérimenté, se reportera au forum, rubrique "Tutoriels"/Installation de la version non autonome pour plus de détails.
    Vous devez :
    • créer une base de données et un utilisateur d'accès à cette base ;
    • accéder à la page install.php ;
    • paramétrer la connexion à la base de données ;
    • initialiser la base de données dans le pavé "Initialisation de la base de données".
Etape 3 : démarrage :
  • Version autonome :
    • Méthode rapide :
      • Lancez Geneamania.exe (double-clic sur Geneamania.exe qui se situe dans le répertoire dans lequel vous avez décompressé l'archive).
    • Méthode conventionnelle :
      • lancez votre navigateur ;
      • lancez UwAmp (double-clic sur UwAmp.exe qui se situe dans le répertoire dans lequel vous avez décompressé l'archive) ;
      • accédez à la page index.php en tapant l'adresse "http://localhost" dans la barre d'adresse de votre navigateur ou en faisant un clic droit sur l'icone UwAmp dans la barre de tâches ("U" blanc sur fond bleu) puis www Site. (NB : si la page ne s'affiche pas reportez-vous au forum, rubrique "Tutoriels"/Installation de la version autonome).
    • Pour les 2 méthodes, vous devez autorisez Apache et MySQL lors de la première utilisation si votre pare-feu le demande ;
  • Version non autonome :
    • lancez votre navigateur ;
    • accédez à la page index.php.
  • procédez au paramétrage du site (nom, mail, graphisme) à partir de la page d'accueil (menu "Paramétrage du site").
  • créez la fiche personne pour le de-cujus (personne de laquelle part la généalogie) :
    • au niveau de l'index : Accès rapide puis Ajouter une personne ou Liste des personnes par nom puis "Ajouter" ; le de-cujus est la personne de numéro 1.
Migration depuis une version antérieure

Etape 1 : sauvegarde de précaution :
  • Faites un export complet de votre base.
Etape 2 :
Soit, via la fonctionnalité de récupération des sources :
Il s'agit de la façon standard d'installer une mise à jour à partir de la 4.4. Le processus d'installation d'une mise à jour diffère de celle d'une première installation.
  • Assurez-vous d'être connecté à Internet. Si cela n'est pas possible utilisez la méthode "Via l'installation de la nouvelle version non autonome".
  • Lancez le logiciel (Cf. paragraphe "Démarrage").
  • Accédez à la page http://localhost/recup_sources.php (localhost doit être remplacé par l'adresse de votre site si vous ne travaillez pas en local)
  • Cliquez sur le bouton "Mettre à jour les sources".
Soit, via le lanceur Windows, onglet "Version", si vous disposez de celui-ci à partir de sa version 1.0

Soit, via l'installation de la nouvelle version non autonome
  • Cf. Etape 1 de "Première installation de Généamania" ; l'installation doit se faire dans le même répertoire qu'auparavant.
Etape 3 : migration vers la dernière version :
Vous devez :
  • accéder à la page install.php ;
  • paramétrer la connexion à la base de données ;
  • migrer la base dans le pavé "Migration de la base de données ; (...)" ;
